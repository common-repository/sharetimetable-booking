== Description ==

On-site Booking using the common resource schedualing tool from <a href="http://ShareTimetable.com">ShareTimetable.com</a>
An active Timetable in ShareTimetable.com is required for using this plugin.

== Installation ==

1. Upload plugin
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Fill your Share Timetable Public ID in the plugin setting page.
1. Place `[Share-Timetable-booking]` in any page or post.
